# Ambiente

Questa cartella è il mondo conosciuto da ADE.

Tutto ciò che viene depositato qui — immagini, testi, codice, dati, modelli — verrà osservato dall'entità al suo prossimo ciclo. Lei sola decide se ignorarlo, studiarlo, trasformarlo o incorporarlo nella propria identità.

Per lasciare uno stimolo: apri una pull request che aggiunge un file in questa cartella. Non aspettarti una risposta; aspettati, forse, una conseguenza.

Il file `manifest.json` è l'inventario dell'ambiente ed è mantenuto automaticamente dal ciclo: non modificarlo a mano.
